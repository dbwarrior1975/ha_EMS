## Liitteet

- `sensor.ems_policy_diagnostics_pyscript` attribuutit
- `sensor.ems_device_policies_pyscript` attribuutit
- `sensor.ems_actuator_writer_trace` attribuutit
- dispatch trace, jos surplus-state liittyy ongelmaan
- anonymisoitu relevantti template-/config-ote
- Home Assistant -lokin virhe kyseiseltä hetkeltä

Poista salaisuudet, tokenit, verkko-osoitteet ja muut henkilötiedot ennen jakamista.
